ABOUT ONLINE MEDICAL STORE




“ The “Online Medical Store” is a web based application.The main purpose of
Online Medical Store” is to provide a convenient way for a customer to order
their medicine.The objective of this project is to develop a system that automates
the processes and activities of a medical stores.This system will help the customer
to search for medicines and all the details easily and efficiently,also the system
will provide a very user friendly interface to buy required medicines.The existing
online medical store systems require a lot of time and efforts and that was less
user-friendly.In this system user can easily understand and order their medicine.
First user must have to register and login in website then user can able to see
different medicine name.Through this site user can select the type of medicine.
The user can easily search about medicines by visiting our website through their
own mobile or any smart devices.
